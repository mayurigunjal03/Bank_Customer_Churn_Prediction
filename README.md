# Bank Customer Churn Prediction

## Project Overview

This project predicts customer churn using Machine Learning and provides interactive business insights through Power BI and Streamlit.

## Business Problem

Customer churn negatively impacts revenue, customer lifetime value, and long-term competitiveness. This project helps identify high-risk customers before they leave.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* Power BI
* Matplotlib

## Machine Learning Models

* Logistic Regression
* Decision Tree
* Random Forest

## Model Performance

| Model               | Accuracy | ROC-AUC |
| ------------------- | -------- | ------- |
| Logistic Regression | 80.95%   | 0.7806  |
| Decision Tree       | 79.00%   | 0.6834  |
| Random Forest       | 86.35%   | 0.8453  |

## Key Churn Drivers
